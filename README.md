Skyfire - Быстрее. Легче. Твой.

Современный, легковесный веб-браузер на базе Python. Skyfire использует движок QtWebEngine (Chromium) для безупречной скорости.

QtWebEngine
Мощь Chromium под капотом. Поддержка всех современных веб-стандартов и высокая скорость рендеринга.

Python Core
